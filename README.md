Cách cài đặt : 
Mở xampp rồi tạo 1 database mới , import database có trong source lên sau đó làm các bước bên dưới
B1 : git clone https://github.com/garnalam/SE2025-16.1.git
B2 : cd SE2025-16.1
B3 : composer install
B4 : npm install --legacy-peer-deps
Sau khi xong mở 2 terminal lên (Nhớ cd lại vào đúng folder vừa clone trên github về)
B5 : Terminal thứ nhất chạy : npm run dev
B6 : Terminal thứ hai chạy : php artisan serve
